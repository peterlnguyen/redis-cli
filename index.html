redis-cli

Common Usage

redis-cli -h endpoint -p port

Examples

redis-cli -h production-replica.asbzmi.3104.usw1.cache.amazonaws.com -p 6379  

redis-cli -h 127.0.0.1 -p 6379

Defaults

  • localhost 127.0.0.1
  • port 6379
  • no password

Common Flags

  • -a    password
  • -v    version
  • -h    help

Install

apt-get install redis-server

Or the current stable version (as of 8-29-14):
